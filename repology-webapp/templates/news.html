2026-04-23

  • Due to increased traffic from misbehaving API clients, the terms of service have been updated: bulk clients are now required to provide a link to their source code repository with an accessible issue tracker.
  • Since there are a lot of clients which query nonexistent project names which look more like package names, we would like to remind you that there is a tool which redirects package name requests to the corresponding project pages, including the API project/ endpoint. Clients trying to query the API by local package names should use it, and the redirects do not count towards the API RPS limit.
  • Due to rising IPv4 address prices and the shortage thereof, and with the majority of Repology traffic being IPv6, we are planning the sunset of IPv4 services in the forthcoming years. A detailed sunset plan will be announced later; in the meantime, all site and API clients are advised to ensure IPv6 connectivity.
